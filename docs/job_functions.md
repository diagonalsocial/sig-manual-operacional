# Funções de trabalho

## Como criar funções de trabalho

Clique no menu `Configurações`, acesse `Funções de trabalho` e siga os passos:

1. Clique em "Adicionar".

2. Defina um nome para a função de trabalho.(O campo descrição é opcional).
